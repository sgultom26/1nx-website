# 1nx.io Website Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the cinematic, bilingual (EN/ID) 1nx.io marketing website per the approved spec.

**Architecture:** Next.js 15 App Router with server-rendered, indexable content as the foundation; a cinematic enhancement layer (Lenis smooth-scroll + GSAP ScrollTrigger pinned/scrubbed sections + an OGL WebGL hero + Framer Motion reveals) mounts client-side over the real DOM. Locale-routed i18n (EN root, `/id/*` mirror) exactly like the sibling 1pay2link site. All motion has `prefers-reduced-motion` and mobile fallbacks.

**Tech Stack:** Next.js 15.5, React 19, TypeScript 5.7 (pinned), Tailwind CSS v4, Framer Motion 12, GSAP 3 + ScrollTrigger, Lenis, OGL, lucide-react, Vitest (logic tests). Fonts via `<link>`: Space Grotesk, Inter, IBM Plex Mono.

## Global Constraints

- **TypeScript pinned to `5.7.3`**, `@types/node` `^22` — TS 6 breaks CSS side-effect imports. Ship `global.d.ts` with `declare module "*.css";`.
- **Tailwind v4** — use `bg-linear-*` not `bg-gradient-*`; tokens in `@theme`.
- **Colour:** dominant white `#FFFFFF` / soft `#F6F8F9`, ink/black `#0A0A0B`; single accent dark cyan `#0E7490` (+ `#0891B2`, bright `#22D3EE` for motion/glow only). No other hues.
- **Type:** Space Grotesk (display), Inter (body), IBM Plex Mono (labels). Near-flat cards (4px radius), hairline borders `#E6E8EB`.
- **Brand name renders lowercase `1nx`.** Slogan: **"One idea. N executions."** Domain `1nx.io`; `NEXT_PUBLIC_SITE_URL` default `https://1nx.io`.
- **i18n:** EN = root, ID = `/id/*` (server-rendered, hreflang+canonical, in sitemap). Code/tech terms stay English.
- **Never run `next dev` and `next start`/`next build` against the same `.next` simultaneously** (shared dir corruption — learned on 1pay2link).
- **Pattern source (read, don't re-derive):** `~/Stuff/Project/1nx/1pay2link/website` (referred to below as `P2L`). Reuse its conventions; restyle to the 1nx theme. Do NOT copy its Clash Display fonts, cyan-free palette, or payment copy.
- **All motion respects `prefers-reduced-motion`** and degrades on touch/small screens; content is identical across states.

## Verification convention (used by every task)

- **Build gate:** `npm run build` completes with no type errors. (Faster inner loop: `npx tsc --noEmit`.)
- **Render check (pages/sections):** after build, grep the prerendered HTML, e.g. `grep -o "<string>" .next/server/app/<route>.html`.
- **Visual check (animation/graphics):** headless screenshot then view it:
  `"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" --headless --disable-gpu --hide-scrollbars --force-device-scale-factor=2 --window-size=1440,900 --screenshot=/tmp/shot.png "http://localhost:3000/<route>"` (run against `npm start`), or build+`next start` then screenshot.
- **Logic check:** `npx vitest run <file>`.
- Commit after each task with the message shown.
